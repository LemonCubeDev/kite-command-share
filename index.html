LemonCube's Kite Command Share

The fast way to install and share complex Kite command flows with a single click.

Tool Overview

This system uses two bookmarklets for a complete sharing workflow:
Generator: Extracts the complex JSON payload from an existing Kite command.
Injector: Automatically installs that extracted JSON payload into a new command.

Installation Instructions: Drag & Drop

To install, simply drag the buttons below up to your browser's bookmarks bar.
At any time, you can reload the page, and both bookmarklets will be deactivated.

KCS Generator KCS Injector

How To Use The Kite Command Share

1.

GENERATE THE SHARING CODE:

  1. Go to the existing Kite command you want to share.
  2. Click the Generator bookmarklet.
  3. Click Save Changes.
  4. The raw JSON code will be automatically copied to your clipboard.
  5. Share the JSON code with the receiving user.
2.

INJECT THE PAYLOAD:

  1. Go to your Kite commands page.
  2. Click the Injector bookmarklet.
  3. A prompt will ask you to enter the JSON sharing code. Paste the JSON string you previously recieved.
  4. The tool is now active! Click Create Command as normal and give it a name and description.
  5. When you click Create Command, the automation will run, copying the command onto your account.
  6. Simply reload the page, and there you go! The command has now been fully copied onto your account.

The bookmarklets above have a single line that uses the full, stable code from generator.js and injector.js. All code is open sourced, so you can have peace of mind when using it.
