The fast way to install and share complex Kite command flows with a single click.

What is this Tool?

This tool installs a small bookmarklet in your browser that intercepts the Kite command creation process. When you create a new command, the tool automatically injects a massive JSON payload (your shared flow) into the new command, saving you dozens of manual copy/paste steps.

Installation Instructions

Follow these two steps to install the KCS bookmarklet in your browser.

1.

Create a New Bookmark:

Go to your browser's bookmark manager (or right-click your bookmarks bar) and create a new bookmark. You can name it anything, like "Kite Injector."

2.

Paste the Code into the URL Field:

Copy the following entire line of code and paste it into the **URL** (or **Address**) field of the new bookmark.

javascript:(function(){var url='https://kcs.lemoncube.dev/kcs_injector.js';fetch(url).then(response=>response.text()).then(script=>{eval(script);}).catch(error=>alert('Error loading LemonCube script: '+error));})();

This tiny script fetches the full, reliable injector code from this hosted page.

How to Use KCS

  1. Go to your Kite App's command page.
  2. Click your **Kite Injector** bookmarklet.
  3. A prompt will appear asking for the **BASE JSON PAYLOAD**. This is the long JSON string of the flow you want to share.
  4. Enter the payload and click OK. The tool is now active!
  5. Go into Kite and **create a new command** as normal (give it a name and description).
  6. When you click "Save," the automation will run: it extracts the new Command ID from the URL and immediately sends a PATCH request to fill it with your large payload!
